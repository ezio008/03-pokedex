# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecturar los siguiente comandos:
```bash
# instalar packages
$ yarn install

# instalar cliente
$ npm i -g @nestjs/cli

# levantar la base de datos
$ docker-compose up -d

# ejecutar en modo desarrollo
$ yarn start:dev
```
